# Skills 介绍文档

本目录存放了专用于特定任务的高级 Skill 定义文件。以下是当前可用 Skills 的详细介绍。

## 1. LayerA-ParagraphMove-Analysis

**文件**: `LayerA-ParagraphMove-Analysis.txt`

### 🎯 目标
专为高水平学术写作与逆向学习设计。对**用户指定的单个或多个论文段落**进行逐句拆解，深入分析作者在每一句话中实施的“说服动作”，并生成可直接迁移的段落级仿写模板。

### ✨ 核心功能
*   **句子级写作动作拆解**: 逐句分析句子作用（如背景、聚焦、对比、防守等），说明其在逻辑构建和读者引导中的具体功能。
*   **写作心法揭秘**: 分析段落整体逻辑流（如铺路、转向、压缩问题），指出关键句及改变语序的后果。
*   **完形填空式段落模板**: 提炼出复现原文逻辑流的英文模板，并详细说明每个占位符的填写要求。

### 🚀 适用场景
*   分析顶级论文的 Introduction、Discussion、Methods 等关键章节。
*   学习如何通过句子编排来增强论文的说服力和逻辑连贯性。
*   需要针对特定逻辑功能（如指出缺口、提出主张）的学术表达模板。

## 2. PDF2MD-Academic

**文件**: `PDF2MD-Academic.txt`

### 🎯 目标
将学术类 PDF（包含复杂的数学公式、图表引用、参考文献等）精确转换为排版整洁、渲染友好的 Markdown 文档。

### 🛠️ 核心规范
*   **公式标准化**: 严格使用 LaTeX 格式，行内公式用 `$...$`，独立公式用 `$$...$$`。
*   **引用统一**: 正文引用与参考文献条目统一使用 `\cite{refn}` 格式，便于链接和阅读。
*   **排版优化**:
    *   “其中”或“where”开头的解释段落首行缩进。
    *   参考文献条目间空行，防止渲染合并。
    *   罗马数字列表自动转换为清晰的 Markdown 列表格式。
*   **算法标准化**: 规范化伪代码块的格式、缩进及关键字大小写。

### 🚀 适用场景
*   需要将 PDF 论文转换为 Markdown 格式以便于阅读、笔记或进一步处理（如 RAG、翻译）。
*   处理包含大量数学公式和引用的学术文献。
